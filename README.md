# Uzbek Transliterator

Fast and accurate transliteration between Cyrillic and Latin Uzbek scripts.

## Usage

```bash
python transliterate.py input.txt -o output.txt -t latin
```

Or via pipe:
```bash
echo "salom" | python transliterate.py -t cyrillic
```

Transliterate Cyrillic Uzbek text to Latin and vice versa.
